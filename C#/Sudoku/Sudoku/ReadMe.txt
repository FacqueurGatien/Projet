Le projet a pour but :
-La réalisation d'une API qui génère aléatoirement des grilles de Sudoku côté back-end.
-La transmission de ces grilles au côté front-end.
-La suppression aléatoire d'un certain nombre de cases.
-L'utilisateur résout la grille.
-Vérification de la validité de la grille une fois complétée.

Technologies utilisées :
-C#
-Entity Framework
-JavaScript (possibilité d'utiliser un framework)
-SQL Server (base de données locale)
-Swagger
La base de données constitue une étape intermédiaire qui ne sera pas nécessaire dans la suite du projet.
L'objectif principal est de faire transiter les grilles du back-end vers le front-end via une API (la base de données est présente pour contrôler cette interaction)
